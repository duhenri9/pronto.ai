# Aula 3: Como fazer perguntas poderosas (Prompting) (10 min)

## Objetivo
Você vai aprender a "conversar" com a IA para tirar o melhor proveito dela.

## Conteúdo

**"Prompt" é a pergunta que você faz para a IA.**

Parece simples, mas a forma como você pergunta muda tudo. É como dar instruções para um ajudante: se você fala "faz aí", ele se perde. Se você fala direitinho, ele faz melhor.

## Os 4 passos para um bom prompt

### 1. Seja claro sobre o que você quer
- Fraco: "Me ajuda com vendas"
- Forte: "Me dá 3 ideias de mensagem para mandar no WhatsApp de cliente que não compra há 30 dias"

### 2. Dê contexto
- Fraco: "Escreve uma legenda"
- Forte: "Escreve uma legenda para Instagram para uma loja de roupas infantis. O tom deve ser carinhoso e divertido. É para o Dia das Crianças."

### 3. Especifique o formato
- Fraco: "Me fala sobre precificação"
- Forte: "Me explica em 3 passos simples como calcular o preço de um bolo caseiro. Usa exemplos com valores em reais."

### 4. Peça para a IA assumir um papel
- Fraco: "Como melhorar meu atendimento?"
- Forte: "Aja como uma consultora de atendimento ao cliente e me diga 3 formas de encantar meus clientes no WhatsApp."

## Exemplo completo
Pergunta fraca: "Me ajuda com marketing?"

Pergunta poderosa: "Você é uma especialista em marketing para pequenos negócios. Me dá 5 ideias de conteúdo para Instagram para uma doceria caseira. O público é mães entre 25-40 anos. O tom deve ser afetuoso. Quero ideias para posts, stories e reels."

## Exercício
Pegue uma dúvida real do seu negócio e transforme em um bom prompt seguindo os 4 passos. Depois pergunte para a Maria e veja a diferença na resposta!
